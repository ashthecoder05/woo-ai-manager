Add Blocko Agent to your store

Choose where you want the assistant to live, then follow the steps.

Plugin-based store (WooCommerce / Shopify / PrestaShop)

WooCommerce: Go to Appearance → Theme Editor → footer.php, paste just before </body>, click Update File.

Shopify: Go to Online Store → Themes → Edit code → theme.liquid, paste before </body>, save.

PrestaShop: Go to Design → Theme & Logo → Advanced customisation, edit footer.tpl, paste before </body>.
API-based store (FastAPI / Flask / Django / Express)

1. Open your main layout/template file
2. Find the closing </body> tag
3. Paste the snippet just above it
4. Redeploy — the ₿ Assistant button will appear

FastAPI/Flask: Add to base.html  |  Django: Add inside {% block extrascripts %}  |  Express: Add to layout.ejs
Custom website

1. Open your admin/checkout page (e.g. admin.html)
2. Find </body> at the bottom
3. Paste the snippet on the line just above it
4. Save and refresh — a ₿ Assistant button appears in the bottom-right

Tip: Add only to your admin page to keep the assistant private.
Add to your store

1. Open your page file in a text editor
2. Find </body> at the bottom
3. Paste the snippet on the line just above it
4. Save and refresh — a ₿ Assistant button will appear in the bottom-right
How to get your bot token: 1. Open Telegram and go to @BotFather
2. Start a chat and type /newbot
3. Follow the prompts to name your bot and choose a username
4. BotFather will send you a token — copy the entire thing (looks like: 1234567890:ABCdef...)
5. Paste it below and click Save & Connect
Bot token saved. Your Telegram bot is connected.

Once added, a ₿ Assistant button appears on your admin page. Customers visiting your store will not see it.
