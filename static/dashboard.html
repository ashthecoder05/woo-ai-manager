Embed Setup
Plugin (WooCommerce / Shopify / PrestaShop)

WooCommerce: Appearance → Theme Editor → footer.php — paste before </body>

Shopify: Online Store → Themes → Edit code → theme.liquid — paste before </body>

PrestaShop: Design → Theme & Logo → Advanced customisation — edit footer.tpl
API (FastAPI / Flask / Django / Express)

1. Open your main layout/template file
2. Find </body> and paste the snippet just above it
3. Redeploy — the ₿ Assistant button will appear

FastAPI/Flask: add to base.html  |  Django: inside {% block extrascripts %}  |  Express: layout.ejs
Custom Website

1. Open your admin/checkout page (e.g. admin.html)
2. Find </body> at the bottom
3. Paste the snippet just above it — save and refresh

Tip: Add only to your merchant admin page, not the public storefront.
1. Open your page file in a text editor
2. Find </body> at the bottom and paste the snippet above it
3. Save and refresh — a ₿ Assistant button appears in the bottom-right
Telegram Setup
Connect your Blocko Agent to Telegram

1. Open Telegram and search for @BotFather
2. Send /newbot and follow the prompts to create your bot
3. Copy the API token BotFather gives you
4. Paste your bot token below and click Save & Connect:

Bot token saved. Your Telegram bot is connected.

5. Your Telegram bot will be live and ready to assist customers

Tip: You can customize the bot's name and avatar in BotFather with /setname and /setuserpic.
